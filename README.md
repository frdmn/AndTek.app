AndTek Agent
============

Handy Cocoa/Objective-C menubar App for Mac OS X to communicate with a AndTek call center.

## Requirements

1. Cisco Unified Communications Manager
1. AndTek call center
1. OS X workstation
1. Xcode

## Installation

* Clone this repo
* Compile from source
* Build the executable via Xcode
